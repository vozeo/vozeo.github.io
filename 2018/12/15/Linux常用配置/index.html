Linux系统配置和软件配置。

Zsh配置

安装zsh shellsudo apt install zsh oh-my-zsh oh-my-zsh-installer

安装oh-my-zshinstall-oh-my-zsh

更改你的默认登入shell为zshsudo chshll -s /bin/zsh vozeo

配置zsh插件：

thefuck
sudo

在zsh配置文件～/.zshrc中把nvim别名改成vim

下载打包好的powerline字体

安装powerline主题

编辑ZSH_THEME="powerline"

Nvim配置

~/.config/nvim/init.vim

设置airline字体

用pip更新python-nvim
pip3 install neovim --upgrade

修改完以后刷新source ~/.zshrc

修改airline主题

在init.vim文件中写let g:airline_theme="deus"
双字宽set ambiwidth=double

在终端使用代理

  1. 安装sudo apt install proxychains
  2. 在/etc/proxychains.conf最下方添加socks5 127.0.0.1 1080
  3. 把其他的注释掉
  4. 在命令前加proxychains
